# Minecraft Skin Generator 

Want a truly unique Minecraft skin? This simple script generates a completely random Minecraft skin in less than a second. With about **10^29,590** possible skins, the chance of someone else generating the same one is effectively zero.

Every run creates a brand-new skin by assigning a random RGB color to every pixel, resulting in a one-of-a-kind design every time.
